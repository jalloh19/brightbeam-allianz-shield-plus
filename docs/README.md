## 🚀 Key Features

- **9-Step Progressive Form:** Optimized for conversion and mobile users.
- **Intelligent Premium Engine:** Real-time calculations for workers and students.
- **Admin Dashboard:** Centralized management and analytics.
- **Security First:** AES-256 encryption for PII and full audit logging.

## 🛠 Tech Stack

- **Backend:** Django 4.2, Django REST Framework
- **Frontend:** Tailwind CSS, Vanilla JavaScript
- **Database:** PostgreSQL
- **Hosting:** Railway
- **Email:** Resend / SendGrid
- **File Upload:** Uploadcare
